# Gajji Sathvik - Portfolio Website

Welcome to my digital portfolio! I am an interactive designer and creative developer who is passionate about creating iconic digital experiences through motion, typography, and creative coding.

## About This Project

This repository contains the source code for my portfolio website. It features a modern, paper-like aesthetic with smooth animations powered by Locomotive Scroll and GSAP, providing an immersive experience for visitors.

## Structure

- `index.html`: The main landing page showcasing my interactive artist persona.
- `about@.html`: Information about me, my awards, publications, and selected clients.
- `work@.html`: A directory of my projects and case studies.
- `work/`: Detailed pages for individual projects.
- `assets/`: Contains the CSS , JavaScript, and images used throughout the site.

## Hosting on Render (Instructions)

This is a static site (HTML/CSS/JS). To host this on Render for free:

1. **Push your code to GitHub** (if you haven't already).
2. Go to [Render Dashboard](https://dashboard.render.com/) and sign in.
3. Click **New +** and select **Static Site**.
4. Connect your GitHub account and select this repository.
5. In the configuration:
   - **Build Command**: Leave blank (not needed for static sites).
   - **Publish Directory**: `.` (the root directory).
6. Click **Create Static Site**.
7. Render will automatically deploy your site and provide you with a live URL .
